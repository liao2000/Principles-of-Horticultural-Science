# 植物荷爾蒙(Phytohormones)

## 定義

### 植物荷爾蒙

+ 需在植物體內合成的有機化合物

+ 僅需低濃度即可調節植物生長與發育

+ 荷爾蒙<u>需由生合成部位移至作用位置</u>使得以發揮作用，此為一種訊號傳遞

  > <u>需由生合成部位移至作用位置</u>：此種說法已過時，比如乙烯直接釋放到空氣中就不是

### 植物生長調節劑

為自然或人工合成的物質，僅需低濃度(ppm)即可調節植物的生長

## 名詞解釋

+ 向頂運輸(Acropetal)：植物體內有機或無機物質由基部運移到頂端的現象，細胞分裂素在根部合成，由根部向上運輸

+ 向基運輸(Basipetal)：與向頂運輸相反，比如生長素在莖頂合成，往根枝條基部和根部運輸

+ 向性(Tropism)：向光性(Photoropism)、向地性(Geotropism)

+ 內生性(Endogenous)：發生在植物內部，例如一些化合物在植物內被合成

+ 外生性(Exogenous)：植物外部所引發的事，例如人工外施化合物而引發植物的反應

+ 生體外(In vitro)：在**人為控制環境下**，將植物的培植體置於試管或燒杯中並供給富含營養養分的培養基，以促進植物的生長

+ 生體內(In vivo)：在活的有基體發生一些反應，如：植物行光合作用

## 生長素 (Auxin)

> + **最早被發現的植物激素**
>
> + **IAA(吲哚-3-乙酸)** 為天然的生長素中最具代表的一種
>
> + IAA 具有**極性**只能向下傳輸
>
> + IAA 可使莖伸長，導致植物的向光性
>
> + IAA 易受藍光及 UV-B 降解，因此通常不直接對植物噴灑 IAA 而是改用 IBA (indole-3-butyric acid)
>
> + 生長素促進頂芽優勢，因此摘除頂芽，可促進側枝數增加
>
> + 高濃度的生長素可促使乙烯的產生

## 細胞分裂素 (Cytokinins, CTK)

> + 主要分佈在**可進行細胞分裂的部位**，如**莖尖**、**根尖**、**未成熟的種子**、**萌發的種子**和**生長中的果實**
>
> + 細胞分裂素在**根尖合成經木質部運到地上部**
>
> + 促進**細胞分裂**、促進**細胞擴大(cell enlargement)** 、促進**芽的分化**
>
> > 當 CTK / IAA 較高時，癒傷組織形成**芽** (受 **Cytokinins** 影響)
> >
> > 當 CTK / IAA 較低時，癒傷組織形成**根** (受 **IAA** 影響)
>
> + 打破種子及芽的休眠
>
> + 促使氣孔打開
>
> + 延緩老化

## 激勃素 (Gibberellins, GA)

> + 激勃素種類繁多(已發現 136 種)，但都是以激勃烷(gibberellane)為骨架的衍生物
>
> + GA 可**促進細胞的延長** (促進節間伸長，節數並不會增加)
>
> + GA 可**促使細胞分裂** (促進維管束形成層的細胞分裂)
>
> + GA 可**打破種子休眠促進發芽**
>
> + GA 可增強 IAA 對養分的動員效應
>
> + GA 可**抑制不定根的形成**

## 乙烯 (Ethylene)

> + 乙烯具有三向反應
>   
>   + 抑制胚莖及根的生長(產生較粗的莖)
>   
>   + 促進莖的肥大
>   
>   + 莖頂鈎的彎曲
>
> + 乙烯促進葉片及果實的離層和老化
>
> + 乙烯促進果實後熟
>
> + 乙烯導致葉片的上偏生長 (epinasty)
>
> + 乙烯可物適應淹水(缺氧)逆境
>
> + 乙烯參與誘導不定根形成
>
> + 打破芽體和種子休眠及促進種子發芽
>
> + 促進鳯梨科開花
>
> + 促進**雌花的花性表現**
>
> + 促進側芽生長

## 離層酸(Abscisic acid, ABA)

> + 合成地方：主要在根尖和葉片細胞的質體內(葉中是葉綠體，根中是澱粉體)
>
> + ABA 成抑制整株植物的生長，也能抑制種子的萌發
>
> + **低濃度下可促進生長**，如在**乾旱逆境**下促使根生長
>
> + 促進植物器官脫落
>
> + 促進種子及芽的休眠
>
> + 維持頂芽優勢，抑制分枝形成
>
> + 促進氣孔關閉

|:----:|:-----------------------:|:-------------:|
| 氣孔打開 | CTK 促進氣孔打開              | ABA 離層酸促進氣孔關閉 |
| 頂芽優勢 | Auxin 促進項芽優勢 ABA 維持頂芽優勢 | 乙烯 促進側芽生長     |
| 種子休眠 | ABA 使子休眠                | GA, CTK 打破休眠  |
| 老化   | 乙烯                      | CTK延緩老化       |
| 不定根  | 乙烯誘導不定根形成及促進側芽生長        | GA 可抑制不定根的形成  |
